# スキル攻撃ダメージ計算式 仕様書

## 1. 計算式表現（数式定義）

最終ダメージ = ベース値 * 基礎_属性バフ枠 * 敵のデバフ枠 * 連撃バフ枠 * トークンバフ枠 * クリティカル補正枠 * 武器相性枠 * 属性相性枠 * 破壊率_特効_ランダム枠

各枠の計算式は以下の通り定義する。

[ベース値]
ベース値 = スキル攻撃力

[基礎_属性バフ枠]
基礎_属性バフ枠 = 1.0 + スキル攻撃力バフ + スキル属性攻撃力バフ + チャージ + アクセサリ + パッシブスキル他 + 心眼バフ + 属性フィールド

[敵のデバフ枠]
敵のデバフ枠 = 1.0 + 防御ダウン + 永続防御ダウン + 属性ダウン + 永続属性ダウン + DP防御ダウン + パッシブスキル他 + 脆弱 + 永続脆弱

[連撃バフ枠]
連撃バフ枠 = 1.0 + 連撃バフ + パッシブスキル他

[トークンバフ枠]
トークンバフ枠 = 1.0 + トークンバフ

[クリティカル補正枠]
・クリティカル発生時
クリティカル補正枠 = 1.5 + クリ威力 + 属性クリ威力 + パッシブスキル他
・非クリティカル発生時
クリティカル補正枠 = 1.0

[武器相性枠]
武器相性枠 = 敵の武器属性相性倍率（斬・突・打）

[属性相性枠]
属性相性枠 = 1.0 + 属性相性値 + 属性ダウン

[破壊率_特効_ランダム枠]
破壊率_特効_ランダム枠 = 現在の破壊率 * (1.0 + 特効倍率) * 乱数

---

## 2. 各項目の補足説明

### ■ ベース値
* **スキル攻撃力**: キャラクターのステータスおよびスキル固有の威力から算出される基礎値。※クリティカル時は敵能力値を「-50」として計算する仕様が別途存在する。

### ■ 基礎・属性バフ枠
* **スキル攻撃力バフ / スキル属性攻撃力バフ**: スキル効果によって上昇する各種攻撃バフ値。
* **チャージ**: チャージスキルの効果値。
* **アクセサリ**: 装備している各種リングやピアスの効果値。
* **パッシブスキル他**: アビリティやスタイルパッシブなどの効果値。
* **心眼バフ**: 心眼によって付与されるバフ値。
* **属性フィールド**: 属性強化フィールド（弱:0.35 / 通常:0.65 / 大:0.80 など）の効果値。

### ■ 敵のデバフ枠
* **防御ダウン / 永続防御ダウン**: 敵に付与されている防御デバフ値。
* **属性ダウン / 永続属性ダウン**: 敵に付与されている属性防御デバフ値。
* **DP防御ダウン**: 敵のDPに対する防御デバフ値。
* **パッシブスキル他**: 敵の弱体化に関わるパッシブ効果値。
* **脆弱 / 永続脆弱**: 敵に付与されている脆弱の効果値。

### ■ 連撃バフ枠
* **連撃バフ**: 連撃数（連撃数上昇バフ）による一撃あたりの重み付け補正値。

### ■ トークンバフ枠
* **トークンバフ**: 特定のスタイルが持つ「トークン」の蓄積数に応じた倍率補正値。

### ■ クリティカル補正枠
* **クリ威力 / 属性クリ威力**: クリティカル時の威力を増加させるバフ・スキル効果値。

### ■ 武器相性枠
* **敵の武器属性相性倍率**: 敵の弱点・耐性（等倍:1.0 / 弱点:1.5 / 耐性:0.5 など）。

### ■ 属性相性枠
* **属性相性値**: 元素属性（火・水・雷・氷・光・闇）の弱点・耐性値。
* **属性ダウン**: エレメンタルダウン等、デバフとは別枠で加算される属性ダウン値。

### ■ 破壊率＆特効＆ランダム枠
* **現在の破壊率**: 敵の現時点の破壊率（100%〜999%を内部的に 1.0〜9.99 として扱う）。
* **特効倍率**: スキルが持つ「HP+50%」などの特定状況下の特効効果値（DP攻撃時はDP特効、HP攻撃時はHP特効を適用）。
* **乱数**: 0.9 〜 1.1 の間でランダムに変動する最終ブレ幅。

## 3. データ定義仕様

ゲーム内およびシミュレーター内部のJSONデータ（`skills.json`, `passives.json` など）で定義されているバフ・デバフの `skill_type` やパラメータと、ダメージ計算式の各枠との対応関係は以下の通りです。

### 1. 基礎・属性バフ枠
計算式：`1.0 + スキル攻撃力バフ + スキル属性攻撃力バフ + チャージ + アクセサリ + パッシブスキル他 + 心眼バフ + 属性フィールド`

| 枠内の項目 | 内部データ定義（`parts[].skill_type` / 条件） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **スキル攻撃力バフ** | `AttackUp`<br>（`elements` が空配列かつ、対象がスキル） | 攻撃力アップ | 通常のスキル攻撃力を上昇させるバフ。回数制限バフ（Count）や専用スキル限定バフ（Only）が存在し、シミュレーターではOnlyのうち最大1つ、Countのうち最大2つの効果値が高いものが優先採用され、永続バフと合算されます。 |
| **スキル属性攻撃力バフ** | `AttackUp`<br>（`elements` に属性元素 `Fire`, `Ice`, `Thunder`, `Light`, `Dark`, `Slash`, `Stab`, `Strike` が含まれる場合） | 属性攻撃力アップ<br>（例: 火攻撃力アップ 等） | 特定の属性（元素属性または武器属性）を伴う攻撃の時のみ適用される攻撃力バフ。属性が一致するスキル発動時にのみ計算されます。 |
| **チャージ** | `BuffCharge` | チャージ | スキル使用時に消費される、次のスキル威力を上昇させるバフ。ダメージ計算の基礎・属性バフ枠に加算されます。 |
| **パッシブスキル他** | `AttackUp` / `AttackUpPerToken`<br>（パッシブスキル・アビリティ由来） | 攻撃力アップ<br>（アビリティ/スタイルパッシブ） | キャラクターの限界突破アビリティやスタイルパッシブ（例：前衛にいるとき攻撃力+10%など）によるバフ値。 |
| **心眼バフ** | `MindEye` | 心眼 | 敵の弱点属性を突いた時のダメージを上昇させるバフ。 |
| **属性フィールド** | `Zone` | フィールド（例: 火属性フィールド 等） | バトルフィールド全体に展開される属性強化フィールド。展開されている属性フィールドと攻撃属性が一致する場合に適用されます（弱: 0.35 / 通常: 0.65 / 大: 0.80など）。 |

---

### 2. 敵のデバフ枠
計算式：`1.0 + 防御ダウン + 永続防御ダウン + 属性ダウン + 永続属性ダウン + DP防御ダウン + パッシブスキル他 + 脆弱 + 永続脆弱`

| 枠内の項目 | 内部データ定義（`parts[].skill_type` / 条件） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **防御ダウン / 永続防御ダウン** | `DefenseDown`<br>（`elements` が空配列かつ、対象が敵全体/単体） | 防御力ダウン | 敵の防御力を下げるデバフ。ターン制限のあるものと、戦闘中永続するもの（永続デバフ）が存在します。 |
| **属性ダウン / 永続属性ダウン** | `DefenseDown`<br>（`elements` に属性元素 `Fire`, `Ice` などが含まれる場合） | 属性防御力ダウン<br>（例: 火防御力ダウン 等） | 敵の特定属性に対する防御力を下げるデバフ。攻撃属性と一致する場合にのみ適用されます。 |
| **DP防御ダウン** | `DefenseDown`のうち、対象がDP限定のもの | DP防御力ダウン | 敵のDPゲージに対する攻撃時のみ有効な防御デバフ。 |
| **脆弱 / 永続脆弱** | `Fragile` | 脆弱 | 敵が弱点属性で受けるダメージを増幅するデバフ。弱点属性攻撃時のみ適用されます。 |
| **永続脆弱** | `Fragile`<br>（効果時間 `Eternal` 等） | 永続脆弱 | 戦闘中永続して付与される脆弱デバフ。 |
| **蝕** | `Undermine` | 蝕 | 敵のデバフ枠（またはデバフ倍率）に関わる特殊な弱体効果。 |

---

### 3. 連撃バフ枠
計算式：`1.0 + 連撃バフ + パッシブスキル他`

| 枠内の項目 | 内部データ定義（`parts[].skill_type`） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **連撃バフ** | `Funnel` | 連撃数アップ | 連撃数上昇バフ。連撃数（回数）と一撃あたりの重み付け（小: 6%, 中: 12%, 大: 25%, 特大: 50% など）から補正値が算出されます。 |

---

### 4. トークンバフ枠
計算式：`1.0 + トークンバフ`

| 枠内の項目 | 内部データ定義（`parts[].skill_type`） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **トークンバフ** | `TokenSet` で獲得したトークンを利用した倍率補正 | トークン上昇 / トークン | キャラクターが蓄積したトークン（最大10個）の数に応じて、対応する特効スキル（`TokenAttack`）などのダメージ計算時に適用される倍率補正。 |

---

### 5. クリティカル補正枠
計算式：
* クリティカル発生時：`1.5 + クリ威力 + 属性クリ威力 + パッシブスキル他`
* 非クリティカル発生時：`1.0`

| 枠内の項目 | 内部データ定義（`parts[].skill_type` / 条件） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **クリ威力** | `CriticalDamageUp`<br>（`elements` が空配列） | クリティカルダメージアップ | クリティカル発生時のダメージ倍率を底上げするバフ。 |
| **属性クリ威力** | `CriticalDamageUp`<br>（`elements` に属性元素が含まれる場合） | 属性クリティカルダメージアップ | 特定の属性でクリティカルが発生した時のみ適用されるクリティカル威力バフ。 |

---

### 6. 属性相性枠
計算式：`1.0 + 属性相性値 + 属性ダウン`

| 枠内の項目 | 内部データ定義（`parts[].skill_type` / 条件） | 日本語表示名 (`STATUS_LABELS`) | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **属性相性値** | `ResistDown` / `ResistDownOverwrite` 等 | 属性耐性ダウン / 属性耐性打ち消し | 敵が元々持っている元素属性（弱点・耐性）の相性倍率。耐性ダウンや属性耐性打ち消しにより変動します。 |
| **属性ダウン** | エレメンタルダウン等、デバフとは別枠のもの | 属性ダウン | 敵のデバフ枠とは別系統で加算される属性ダウン効果。 |

---

### 7. 破壊率_特効_ランダム枠
計算式：`現在の破壊率 * (1.0 + 特効倍率) * 乱数`

| 枠内の項目 | 内部データ定義（`parts[].multipliers`） | 日本語表示名 | 補足説明・適用仕様 |
| :--- | :--- | :--- | :--- |
| **特効倍率** | `parts[].multipliers.hp`（HP特効）<br>`parts[].multipliers.dp`（DP特効）<br>`parts[].multipliers.dr`（破壊率特効） | 特効倍率（例: 対HP+30% 等） | スキル自体に設定されている、特定の状況（HP攻撃時またはDP攻撃時）においてダメージが増幅される倍率。 |
