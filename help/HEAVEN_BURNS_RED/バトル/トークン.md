# ヘルプ：HEAVEN BURNS RED > バトル > トークン

## 基本情報

- **カテゴリ**: HEAVEN BURNS RED > バトル

## ヘルプ記載内容
トークン
バトル中に付与される効果の1つです。主にスキルで付与することができ、付与数に応じて威力が上がるスキル攻撃などがあります。※トークンは最大で10まで溜まります

## シミュレーター実装情報
- **内部名**: `TokenSet`
- **条件判定名**: `OverDriveGauge() >= 0` などを利用
