## ヘルプ記載内容

**魔界騎兵起動ボタン（特殊ギミック「魔界騎兵（メカデモンガー）」）**

SS＜レゾナンス＞[誇り高き魔王の凱旋] 山脇・ボン・イヴァール が持つ特殊ギミックです。

バトル開始時に画面上に **「魔界騎兵起動ボタン」** が出現します。このボタンを押すと、**ターンを消費せず即座に**大怪人メカデモンガーが起動して攻撃を行います。

- **出現条件**: バトル開始時（パッシブ「魔界騎兵起動！」Lb0 で自動付与）
- **使用可能回数**: 合計3回 / ただし出撃中（連戦コンテンツ等）は初回出撃時に3回付与されたら、その後復活しない（本シミュレーターではスコープ外）
- **ターン進行**: ボタン押下はターンを消費しない（通常のターンと独立して即発動）
- **同一ターン内複数回発動**: 残り回数があれば同じターン内に複数回押すことも可能

### 魔界騎兵起動の効果（スキルID: 46003117）

- 敵全体に **6連撃**（斬属性）[貫通クリティカル]、SP消費なし
- 攻撃前に **「魔王への畏怖」** を付与（敵の全能力-100、次の敵ターン終了まで持続）
- ※ 3回まで使用可能 / 同じターンに複数回発動も可能

### 山脇様のしもべ状態について

「ごきげんダンス」や「魔王様降臨！」によって付与される特殊状態。
「いざ進軍！」「魔王軍の大攻勢！」「ギガビッグバン」の発動条件に関わります。
パーティ全体（6人）がしもべ状態の場合、「ギガビッグバン」が使用可能になり「魔王軍の大攻勢！」でODゲージ+100%が発動します。

**UI Next Phase 1 の解決方針**:

本シミュレーターでは、`note` 文字列へ自動追記せず、`LightweightReplayTurn.operations` を正本として扱います。

1. **ボタン表示条件**: `SS＜レゾナンス＞[誇り高き魔王の凱旋] 山脇・ボン・イヴァール` が所属しているときだけ、未コミット行に `騎兵起動` ボタンを表示します。
2. **記録方法**: ボタン押下で `ActivateMakaiKihei` operation を pending chip として追加し、commit 時に `ReplayScript.turns[i].operations` へ保存します。
3. **取消し方法**: メモ欄上部の operation chip 右端 `×` で取消します。committed 行でも削除可能で、削除時はその turn から再計算します。
4. **残回数管理**: `SpecialCommandCountUp.power[0] = 3` を初期上限とし、battle state へ持たず committed replay turns と current pending operations の合計から残回数を導出します。
5. **ターン非消費**: `騎兵起動` は通常 slot action とは別の special operation として扱い、turn を進めず、行動枠や SP を消費しません。
6. **Phase 1 で反映するもの**: 埋め込み skill `BIYamawakiSkill55b` を preview 用に解決し、OD 上昇のみを state に反映します。
7. **Phase 1 で未反映のもの**: ダメージ、敵 HP/DP 変動、`FearOfDevil`、被ダメ・record 明細は後続フェーズで扱います。

補足:

- 同一 turn 内に複数回追加できますが、残回数 3 を超えて積むことはできません。
- 自由記述メモは別 textarea として保持し、operation chip の追加・削除では変更しません。
