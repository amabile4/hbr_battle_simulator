# ヘルプ：HEAVEN BURNS RED > バトル > 速弾き

## 基本情報

- **URL**: なし (コラボレーション限定ステータス)
- **カテゴリ**: HEAVEN BURNS RED > バトル

## ヘルプ記載内容

**速弾き (Shredding)**
「Angel Beats!」コラボスタイルのSS [Stir Soul Song] 芳岡ユイ が付与可能な特殊なバフ状態です。

- **効果**: 速弾き状態のメンバーは、所持している全スキルが**【SP0以上で使用可能】**になります。
  ※ つまり、スキル発動に必要な消費SPに達していなくても（SPが0以上であれば）スキルを使用できます。
- **制限事項**:
  - 「ダメージスキル(SP全消費)」など、一部の特殊なSP消費のスキルにはこの効果が適用されません。
  - 消費SP以外の「スキル発動条件（OD中、特定の追加バフ状態など）」は通常通り満たす必要があります。

## シミュレーター実装情報

- **内部名**: `Shredding`
- **条件判定名**: `IsShredding()` 等を想定

### 実装メモ

JSON上の解析において、「速弾き」ステータスは以下のように定義・付与されます。

- **付与元のスキル**: 「かき鳴らせキラーチューン」(ID: 46040604)
- **バフ効果定義 (parts)**: 
  - `skill_type: "Shredding"`
  - `target_type: "AllyAll"` (味方全体)
  - `exitCond: "PlayerTurnEnd"`, `exitVal: [3, 0]` (ターン制: 3ターン持続)

**シミュレーターでの解決 (Resolution) と評価**:
本シミュレーターのターン進行およびコマンド選択バリデーションにおいて、キャラクターが `Shredding` バフを保持している場合、以下の処理フックが必要です。

1.  **コマンド選択UI**: スキル群の活性/非活性(`disabled`)判定ループにおいて、通常は `currentSp < skill.spCost` であれば非活性としますが、`Shredding` 状態であれば `currentSp >= 0` を条件として活性化します。
2.  **SP消費処理**: スキル本来のSPコスト分はきっちり消費されます。つまり、現在SPが消費SPを下回っている場合、**SPはマイナスに突入**します（例: 現在SPが7の時に、消費SP18の技を使用するとSPは -11 になる）。
    - シミュレーター実装時は、現在SPからそのままコストを引き算し、マイナス値を許容するロジックとする必要があります。

※ ヘルプ文面の「SP0以上で使用可能になる」という表現から、本来のコストを支払う（不足分は無視されて発動できる、または現在SPを全消費して発動する）挙動が推測されます。
