# ヘルプ：HEAVEN BURNS RED > バトル > 属性ブレスレット

## 基本情報

- **カテゴリ**: HEAVEN BURNS RED > バトル
- **データソース**: `json/accessories.json`

## 概要

属性ブレスレットは、通常攻撃を特定の元素属性へ変えるアクセサリ群である。

- `炎撃のブレス` → 通常攻撃が火属性に
- `氷撃のブレス` → 通常攻撃が氷属性に
- `雷撃のブレス` → 通常攻撃が雷属性に
- `光撃のブレス` → 通常攻撃が光属性に
- `闇撃のブレス` → 通常攻撃が闇属性に

`accessories.json` には複数個体が存在するが、シミュレーター上で重要なのは
「通常攻撃をどの属性に変えるか」である。

## シミュレーター実装情報

- `Party Setup` の `火 / 氷 / 雷 / 光 / 闇` selector は、実アクセサリ個体選択ではなく
  属性ブレスレット種別の代表設定として扱う
- session / setup snapshot では `setup.normalAttackElementsByPartyIndex` に
  `{"0":["Ice"]}` のような形で保存する
- runtime では `CharacterStyle.normalAttackElements` に渡し、通常攻撃の属性参照へ使う
- 通常攻撃の **OD hit 数** は従来どおり `2.5%` 固定で、属性参照だけが
  ブレスレット要素へ差し替わる
- Eシールド判定も同じ `normalAttackElements` を参照するため、通常攻撃が
  Eシールド対応属性になっていれば `1 hit = 1 count` で減算される

## 現時点の非対応範囲

- アクセサリ文言に含まれる `通常攻撃の攻撃力+100〜200%` / `+150%` / `+200%`
  のダメージ補正は、今回のスコープでは未対応
- 実アクセサリ個体名ごとの UI 選択や、複数属性の通常攻撃化は扱わない
