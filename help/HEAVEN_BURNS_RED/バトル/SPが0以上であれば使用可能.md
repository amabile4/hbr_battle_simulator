# ヘルプ：HEAVEN BURNS RED > バトル > SPが0以上であれば使用可能

## 基本情報

- **URL**: なし (ゲーム内共通仕様)
- **カテゴリ**: HEAVEN BURNS RED > バトル

## 概要

スキルの説明文に「SPが0以上であれば使用可能」と記載されているものは、本来の消費SPに満たない状態であっても、現在SPが0以上（つまりマイナスでなければ）使用することができます。

- **SPのマイナス突入**: コスト分はそのまま消費されるため、現在SPがコストを下回る状態で発動した場合はSPがマイナスに突入します。
  - 例: 現在SPが7の時に、消費SP18の対象スキルを使用すると、SPは `-11` になります。
