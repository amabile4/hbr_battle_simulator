# ヘルプ：HEAVEN BURNS RED > バトル > SPが0以上であれば使用可能

## 基本情報

- **URL**: なし (ゲーム内共通仕様)
- **カテゴリ**: HEAVEN BURNS RED > バトル

## 概要

スキルの説明文に「SPが0以上であれば使用可能」と記載されているものは、本来の消費SPに満たない状態であっても、現在SPが0以上（つまりマイナスでなければ）使用することができます。

- **SPのマイナス突入**: コスト分はそのまま消費されるため、現在SPがコストを下回る状態で発動した場合はSPがマイナスに突入します。
  - 例: 現在SPが7の時に、消費SP18の対象スキルを使用すると、SPは `-11` になります。

## JSON 識別方法

**`is_adv: true` かつ `sp_cost > 0`** が完全な識別子です。

- `skills.json` において `is_adv: true && sp_cost > 0` の全37件が「SP0以上であれば使用可能」条件を持つ（例外なし）
- `is_adv: false` のスキルでこの条件を持つものは 0 件
- `cond` フィールドには記載なし（`desc` テキストのみ）

シミュレータ実装では `hbr-data-store.js` のスキルロード時に `is_adv: true && sp_cost > 0` を検出して `cond: 'Sp()>=0'` を付与する方針。
