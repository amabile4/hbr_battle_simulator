# 目的
`dimension_battle.json` というデータを読み込み、ユーザーが「恒星戦」を選択し、その中に含まれる「惑星戦ごとのエンチャント」から取得したいものをチェックボックスで選択できるUIを構築してください。

# 達成したいUI要件
1. **恒星戦の選択**: 
   画面上部にリストボックス（またはプルダウン）を設置し、「第1回 恒星戦」「第2回 恒星戦」... というように大会を選択できるようにしてください。
2. **惑星戦エンチャントの表示**: 
   リストボックスで恒星戦が選択されたら、その下に「惑星戦1」「惑星戦2」「惑星戦3」「惑星戦4」それぞれのセクションを表示してください。
3. **チェックボックスの設置**: 
   各惑星戦の横または下に、該当する惑星戦で取得可能な「エンチャント効果（テキスト）」をラベルとしたチェックボックスを設置してください。

# データ構造仕様 (dimension_battle.json)
対象とするJSONデータは、以下のような階層構造になっています。

- **Root (配列)**:
  JSONのルートは配列になっており、配列の1つの要素が「1回の恒星戦（大会）」のオブジェクトを表しています。
- **恒星戦オブジェクト**:
  - `label` (String): `"dimension_001"` のような値が入っています。この末尾の数字を利用してUI上の「第N回 恒星戦」というラベルを生成してください。
  - `satellites` (配列): その恒星戦に紐づく惑星戦のデータが格納されている配列です。基本的に4つの要素（惑星戦1〜4）が含まれています。
- **惑星戦オブジェクト (satellites配列内の要素)**:
  - 順番（インデックス0〜3）がそのまま「惑星戦1〜4」に対応しています。
  - `enchant` (オブジェクト): 惑星戦で得られるエンチャントのデータです。
    - `desc` (String): UIのチェックボックスの横に表示すべきエンチャント効果のテキストデータです。（例: `"戦闘開始時ODゲージ+200%"`）※値の欠損やnullはありません。

# JSON構造のサンプル
[
  {
    "id": 191000001,
    "label": "dimension_001",
    "satellites": [
      {
        "btl": "Dimension_01_Satellite01",
        "enchant": {
          "desc": "戦闘開始時ODゲージ+200%"
        }
      },
      ... // (計4要素)
    ]
  },
  ...
]

# 実装上の制約・ルール
- 使用するフレームワークや言語にあわせて適切な状態管理（選択された恒星戦のid保持、各エンチャントのON/OFF状態の保持）を実装してください。
- 初期表示時は、もっとも新しい（配列の最後の要素、あるいは第1回）恒星戦が選択された状態にしてください。
- 実装コードの提出をお願いします。

## Stage Setup 現在対応（2026-04-22）

`ui-next` の Stage Setup では、恒星戦プリセットから選択したエンチャントを上段入力へ転記しつつ、
一部の効果は read-only の「有効なプリセット効果」として保持・表示する。

### 手動入力欄として使えるもの

- 毎ターンOD±n%
  - `turnlyOdGauge` として save/load される
  - T1 初期表示時点から 1 回適用し、以後も各ターン開始時に同値を加算する
  - `ODゲージ上昇量+20%` の drive bonus 補正枠とは独立

### 上段入力へ直接転記されるもの

- 戦闘開始時ODゲージ±n%
- 戦闘開始時SP+n
- 防御力50%アップ
- デバフ無効1回付与
- 毎ターンOD±n%
- 毎ターンSP+n
- 毎ターン前衛のSP+n
- 毎ターン後衛のSP+n

### 「有効なプリセット効果」で保持・save/load され、runtime でも反映されるもの

- ODゲージ上昇量+20%
  - Party Setup の `ODピアス` と同じ補正枠に加算する
  - 通常攻撃 / pursuit OD / `enemy od_rate` 解釈には追加変更しない
- ターン開始時ダウンターン中の敵がいるとSP+2
- 敵を倒したとき敵1体につき味方全体のSP+1
- ターン開始時SP0未満の前衛の味方のSP+2
- ターン開始時SP0未満の後衛の味方のSP+2

### 未対応のまま残しているもの

- 回復スキルの効果量+50%
- 破壊率上昇量+100%
- 行動開始時ダウンターン中の敵がいると各種火力/破壊率補正
- 毎ターンDP+10%
- ターン開始時スタン状態の味方のスタン解除
- 3ターン味方全体の防御力+30%
