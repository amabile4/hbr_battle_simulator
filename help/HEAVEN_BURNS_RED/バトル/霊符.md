## ヘルプ記載内容

霊符
バトル中に付与されるデバフ効果の1つです。霊符レベルに応じて全能力が減少します。霊符状態で攻撃を受けたときに霊符レベルが+1されます。※Hit数に関わらず1回の攻撃で霊符レベルが+1されます。※敵の行動終了時にレベルは0にリセットされます※霊符状態を重複させることはできません※全能力を下げる効果を重ねがけした場合、ダメージ算出において効果値の高いほうで算出されます

> **主語の補足（紛らわしい箇所の明確化）**: 霊符状態は**敵に**付与されるデバフです。「攻撃を受けたとき」の主語も**敵**（霊符状態になった敵）であり、プレイヤーパーティーが敵を攻撃するたびに霊符レベルが上昇します。「敵の行動終了時」のリセットも同様に敵ターンを指します。

## シミュレーター実装情報

- **内部名**: `Talisman`

### 詳細仕様 (解析情報)

#### 貼ったりましょう！ (スキル等により付与)

- バトル開始時 敵に霊符状態(ターン永続/解除不可)を付与する

#### 全体 霊符 (Talisman)

- 戦闘開始時、**敵に**レベル0の霊符デバフを付与する。
- 霊符レベルが1上がるごとに、**敵の**全ステータスが10減少する（最大レベル: 10）。
- **霊符状態の敵が**プレイヤーの攻撃を受けるごとに、霊符レベルが1上昇する（1攻撃=+1、Hit数不問）。
- **敵の**ターン終了時にレベルは0にリセットされる。
- ※ 同種の効果とは重複しない。
- ※ 効果値が高いものが優先される。
- ※ 解除不可（Eternal）。

### 実装メモ

霊符は敵共通の field-like state として `enemyState.talismanState` に保持します。`Talisman` effect は battle-start 初期付与、timing level-up、`AdditionalHitOnExtraSkill`、プレイヤー攻撃被弾時の +1、敵ターン終了時 reset を同一 helper 経路で更新します。

現行シミュレーターは「霊符レベルに応じた全能力低下」を `damageContext.enemyAllAbilityDownByEnemy` と UI/record へ露出するところまでを正本とします。専用の数値ダメージエンジンは別途持たないため、霊符による能力低下量は enemy popup / field chip / char detail / action record で確認します。

### 実装調査メモ（2026-04-10）

#### データ構造

`skills.json` / `passives.json` での `skill_type` は `Talisman`。`power[0]` の値で動作が分岐する。

| `power[0]` | `value[0]` | 動作 |
|-----------|-----------|------|
| `0` | `0` | **初期付与**: 霊符状態を active=true、level=0 で敵に付与する |
| `2`（正の値） | `1` | **レベル増加**: `value[0]=1` は「霊符状態のときのみ」条件。レベルを +N する（最大10） |

#### スキルデータの実例

- `貼ったりましょう！`（skills.json, id: 46401601）: `is_restricted=1`, `passive.timing=OnBattleStart`, `parts[0]={skill_type:Talisman, power:[0,0], value:[0,0]}`
  - → バトル開始時に敵へ霊符を初期付与するトリガースキルパッシブ
- `恐怖の叫び`（passives.json → styles.json `passives[]`）: `timing=OnFirstBattleStart`, `parts=[{AdditionalHitOnExtraSkill}, {Talisman, power:[2,0], value:[1,0]}]`
  - → EXスキル使用後（`AdditionalHitOnExtraSkill` トリガー条件）に霊符レベル+2。現行 simulator では dedicated runtime/UI test 済み
- `ハロウィンナイト`（passives.json → styles.json `passives[]`）: `timing=OnPlayerTurnStart`, 条件「霊符状態かつ氷属性スタイル4体以上」、`parts=[{Talisman, power:[2,0], value:[1,0]}]`
  - → ターン開始時に条件満たせば霊符レベル+2
