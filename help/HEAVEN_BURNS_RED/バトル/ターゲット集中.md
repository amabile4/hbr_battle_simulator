# ヘルプ：HEAVEN BURNS RED > バトル > ターゲット集中（かばう / Cover）

## 基本情報

- **内部名**: `Cover`
- **UI表示名**: かばう
- **カテゴリ**: HEAVEN BURNS RED > バトル

## 概要

**Cover（かばう）** は、使用者自身に付与される自己バフ状態です。

### 効果

1. **全体攻撃の集中**: 敵が全体攻撃をしてきた場合、通常は前衛メンバー全員（最大3人）に1回ずつ攻撃が当たるところを、**Cover 状態の使用者1人が3回分の攻撃をすべて受ける**。
2. **単体攻撃の強制ターゲット変更**: 単体攻撃で他のメンバーが攻撃対象に選ばれた場合でも、**強制的に使用者をターゲットに変更する**。

### 被弾パッシブへの影響

- 被弾時に発火するパッシブ（「被弾時に〇〇する」系のスキル条件）の発火回数は、実際に攻撃を受けた回数に連動する。
- **Cover 状態で全体攻撃を受けた場合、被弾パッシブは3回発動する。**

### 持続期間

- `exitCond: EnemyTurnEnd` — 敵のターン終了時に解除される（1回の敵ターンのみ有効）。
- `power: [2, 3]` — スタイルのパラメータ（SPR/CON）に応じて2〜3ターン分の持続になる可能性あり（実機要確認）。

### Attention（注目）との違い

| 効果 | 対象 | 付与者 |
|------|------|--------|
| **Cover（かばう）** | 使用者自身 → 攻撃を自分に集中させる | 味方スキルで自己付与 |
| **Attention（注目）** | 敵 → その敵を攻撃対象に強制 | 味方スキルで敵に付与 |
| **Provoke（挑発）** | 敵 → その敵しか狙えなくする | 味方スキルで敵に付与 |
