## ヘルプ記載内容 (ゲーム内仕様)

超越ゲージは、超越ゲージを所持するアビリティを持つスタイルを編成したとき、戦闘画面に出現する属性別の特殊ゲージです。

- **出現条件**: `火の律動` など、超越ゲージを所持するアビリティを持つスタイルを編成すると、対応属性のゲージが出現する
- **初期値**: 部隊の該当属性スタイルの人数に応じて初期値が上昇する
  - `該当属性スタイル人数 x 15%`
- **上昇条件**: 該当属性のスタイルが1回行動するごとに `+4%`
  - 通常攻撃・スキル・追撃を含む
  - 非該当属性スタイルが該当属性の通常攻撃やスキルを使用しても加算されない
- **100%到達時**: ODゲージが `100%` 上昇し、味方の該当属性スタイルが `バースト` 状態へ移行する

バースト状態では、以下の効果が付与されます。

- 攻撃力 `300%` UP
- 破壊率上昇量 `10%` UP
- 攻撃力アップスキル効果量 `20%` UP
- デバフスキル効果量 `20%` UP
- クリティカル確定
- クリティカルダメージ `100%` UP [単独発動]
- 上限を突破して破壊率UP
  - 最大 `300%`
  - 超ダウン・強ブレイクとは重複しない

### シミュレーターでの解決 (Resolution) と評価

本シミュレーターでは、超越ゲージを `turnState.transcendence` として管理します。

実装済み:

1. 超越アビリティ所持スタイルの編成時に、該当属性ゲージを初期化する
2. 初期値を `該当属性スタイル人数 x 15%` で計算する
3. 該当属性スタイルの action/cast ごとに `+4%` する
4. 該当属性スタイルの追撃を 1 action/cast として `+4%` する
5. 非該当属性スタイルの属性付き通常攻撃・属性付きスキルでは加算しない
6. 100%到達時に ODゲージを `100%` 加算する

  - 攻撃力 `300%` UP
  - 破壊率上昇量 `10%` UP
  - 攻撃力アップスキル効果量 `20%` UP
  - デバフスキル効果量 `20%` UP
  - クリティカル確定
  - クリティカルダメージ `100%` UP [単独発動]
  - 破壊率上限突破

これらのバースト効果は `json/transcendence_rule_overrides.json` にデータとして定義されていますが、現在の戦闘計算エンジンではまだ適用処理が実装されていません。

## 実装メモ

超越ゲージの加算は、攻撃の属性ではなく「行動したスタイルの属性」を基準に判定します。

例:

- 闇属性スタイルの二階堂三郷が行動した場合、闇超越ゲージは `+4%`
- 闇属性スタイルのビャッコがラッシュで同一スキルを2回 cast した場合、闇超越ゲージは `+8%`
- 闇属性スタイルの山脇が追撃した場合、闇超越ゲージは `+4%`
- 非闇属性スタイルのクイーンが闇属性扱いの通常攻撃で Dark Eシールドを減らしても、闇超越ゲージは増加しない

対象 replay では、追撃なしの場合は #2 終了時点で闇超越ゲージ `99%` となり、山脇の追撃が発生した場合に `100%` へ到達します。
