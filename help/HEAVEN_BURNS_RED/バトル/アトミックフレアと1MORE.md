# ヘルプ：HEAVEN BURNS RED > バトル > アトミックフレアと1MORE

## 基本情報

- **URL**: なし (ゲーム内共通仕様)
- **カテゴリ**: HEAVEN BURNS RED > バトル

## 概要

`慧眼の女教皇` クイーンの `アトミックフレア` は、Eシールドを持つ敵に対して特殊な弱点命中として扱う。

- `アトミックフレア`: SP16 / 打 / EXスキル / 単体5hit
- 効果: 敵に特大ダメージ `[貫通クリティカル][Eシールド属性無視][対DP+100%]`
- 追加効果: このスキルでブレイクしたとき ODゲージ +100%
- 追加ターン中: 消費SP0
- `1MORE`: 自身のアクティブスキルで敵の弱点をついたとき、自身に追加ターンを付与する。ただし追加ターン中は付与しない。

Eシールドが存在する敵に `アトミックフレア` を使用した場合、`Eシールド属性無視` により、Eシールドの属性に関係なく「弱点を突いた」と同じ扱いになる。

そのため、Eシールド敵への解決順は次の通り。

1. `アトミックフレア` を使用する
2. active な Eシールドがある場合、Eシールド属性を無視して hit 数ぶん current を減らす
3. 同じ action を弱点命中扱いにする
4. `1MORE` の `AdditionalHitOnWeak` が発火する
5. クイーン自身に追加ターンを付与する
6. 追加ターン中に再度 `アトミックフレア` を使う場合、消費SPは0になる

### シミュレーターでの解決 (Resolution) と評価

シミュレーターでは、`IgnoreEShieldElement` を持つアクティブ攻撃が active な Eシールド敵に命中した場合、その action を Eシールド減算対象かつ弱点命中扱いにする。

評価条件:

- 対象敵に active な `eShieldState` が存在する
- 使用スキルの有効 part に `IgnoreEShieldElement` がある
- 使用スキルが damage part を持つ
- action の対象にその敵が含まれる

この条件を満たすと、通常の属性一致がなくても Eシールドを減らし、`actionHitsEnemyWeakness()` も true として扱う。

対象外:

- Eシールドが存在しない敵
- Eシールドが inactive な敵
- `IgnoreEShieldElement` を持たない攻撃
- 追加ターン中の `1MORE` 再付与
