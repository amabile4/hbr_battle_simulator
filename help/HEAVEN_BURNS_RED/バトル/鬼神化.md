# ヘルプ：HEAVEN BURNS RED > バトル > 鬼神化

## 基本情報

- **URL**: なし（キャラクター固有ギミック）
- **カテゴリ**: HEAVEN BURNS RED > バトル

## ヘルプ記載内容

**鬼神化（きしんか）**

SS＜レゾナンス＞[希望の暁] 手塚 咲 が持つ特殊ギミックです。

バトル中、画面上に出現する **「鬼神化」ボタン** を押すことで、手塚咲が **鬼神状態** に変化します。鬼神状態は **3ターン** 持続し、その間は圧倒的な攻撃力と手数で戦います。

### 鬼神化ボタンの使用条件

- 手塚咲が **鬼神化中でない**（isReinforcedMode = false）
- 手塚咲が **行動不能中でない**（鬼神化後の待機ターンでない）

### 鬼神状態中の効果（3ターン持続）

| 効果 | 内容 |
|------|------|
| **スキルSP消費 0** | 手塚咲の全スキルのSP消費が0になる（例外あり）※1 |
| **連撃数+3** | 全スキルのヒット数が +3 される（大ダメージ）|
| **心眼** | 弱点属性に与えるスキル攻撃のダメージが増加する（増加量は power 値に依存）→ 詳細は[心眼.md](心眼.md) |
| **スキル攻撃力+300%** | パッシブ「鬼道」により、鬼神状態中はスキル攻撃力+300%（Lb0）|

※1 SP消費0にならない例外：
- SP全消費スキル（sp_cost = -1）
- EP消費スキル
- 士気（Morale）消費スキル
- やる気（Motivation）消費スキル

### 鬼神化時の即時効果

- **オーバードライブゲージ +15%**（ボタン押下時に即時付与）

### デメリット：鬼神化後の行動不能

鬼神状態が終了（3ターン経過）すると、自動的に **1ターン行動不能** になります。

```
通常状態
  → [鬼神化ボタン押下] → 鬼神状態（残り3ターン）
  → 1ターン経過  → 鬼神状態（残り2ターン）
  → 1ターン経過  → 鬼神状態（残り1ターン）
  → 1ターン経過  → 鬼神状態終了 → 行動不能（残り1ターン）
  → 1ターン経過  → 通常状態に復帰
```

### 実装上のステート管理

`CharacterStyle` インスタンスが以下のフィールドで状態を保持します：

```javascript
{
  isReinforcedMode: boolean,        // 鬼神化中フラグ
  reinforcedTurnsRemaining: number, // 残りターン数（初期値3）
  actionDisabledTurns: number,      // 行動不能ターン数（鬼神化終了後に1が自動付与）
}
```

鬼神化中の付与ステータスエフェクト（`statusEffects` 配列）：

```javascript
[
  {
    statusType: 'Funnel',              // ヒット数+3（大ダメージ）
    exitCond: 'PlayerTurnEnd',
    remaining: 3,
    power: 3,                          // ヒット+3
    sourceSkillLabel: 'STezukaKishin',
    sourceSkillName: '鬼神化',
  },
  {
    statusType: 'MindEye',             // 心眼: 弱点属性ダメージ増加（増加量は power 値に依存、詳細は心眼.md）
    exitCond: 'PlayerTurnEnd',
    remaining: 3,
    power: 1,                          // 1 = JSON の MindEye power[0] / power[1] 値をそのまま適用（ロール・スキルLvで変動）
    sourceSkillLabel: 'STezukaKishin',
    sourceSkillName: '鬼神化',
  }
]
```

### 鬼神化ボタンのUIロジック

```javascript
// dom-adapter.js
updateKishinkaButtonState() {
  button.disabled = tezuka.isReinforcedMode ||
                    tezuka.actionDisabledTurns > 0;
}
```

状態表示（UI上のバッジ）：

- 鬼神化中: **「鬼神化中: 残りNターン」**

### ターン進行との関係

- **鬼神化ボタン押下自体はターンを消費しない**（通常の行動選択とは独立）
- **鬼神状態のターンカウントダウン**は `updateReinforcedModeStateAfterTurn()` でターン終了時に処理
- Extraターン実行中も手塚咲が Actionable であれば reinforcedTurnsRemaining がカウントダウンされる

**シミュレーターでの解決 (Resolution) と評価**:

本シミュレーターでは以下の処理が実装済みです：

1. **鬼神化ボタン**: `[data-action="kishinka"]` の専用ボタン。使用条件に応じて `disabled` を制御。
2. **SP消費0**: スキル選択UIおよびSP計算ロジックで `STezuka + isReinforcedMode` を判定して消費0に。
3. **ヒット数+3**: `statusEffects` の `Funnel` エントリを参照してヒット数計算に加算。
4. **ターン経過管理**: `tickReinforcedModeTurnIfActionable()` でターン終了時に reinforcedTurnsRemaining をデクリメント。0になると `isReinforcedMode = false` かつ `actionDisabledTurns = 1` を自動セット。
5. **行動不能**: `actionDisabledTurns > 0` のとき、スキル選択を「行動なし」のみに制限。
6. **OD+15%**: ボタン押下時に `state.turnState.odGauge += 15` を即時反映。
