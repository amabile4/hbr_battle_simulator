# ヘルプ：HEAVEN BURNS RED > バトル > 士気

## 基本情報
- **URL**: https://wfs-heaven-burns-red.zendesk.com/hc/ja/articles/30187153931161-%E5%A3%AB%E6%B0%97
- **カテゴリ**: HEAVEN BURNS RED > バトル

## ヘルプ記載内容
士気
バトル中に付与されるレベルバフ効果の1つです。レベルに応じてDPとHPを除く全能力とスキル攻撃力が上昇します。※闘志と重ねがけした場合、全能力上昇においては効果値の高い方の効果が優先されます。

## シミュレーター実装情報
- **内部名**: `Morale`
- **条件判定名**: `CountBC(IsPlayer() == 1 && IsCharacter(EAoi) == 1 && MoraleLevel() >= 6)>0` などを利用
