## 2. 最終仕様案 vNext
### Must
- バトル状態管理
  - バフ/デバフ有効期間（ターン/回数/永続）
  - OVERDRIVE中のターン減算例外
  - フィールド/陣の単一スロット制御
  - BREAK/強ブレイク/破壊率/トークン(上限10)
- 進行・更新
  - 日次 `04:00`、月次 `毎月1日 04:00` の更新基準
  - 連戦系コンテンツの6要素持越し
- アカウント/課金
  - 引き継ぎプラットフォーム別有償クォーツ移行マトリクス
  - 年齢帯別課金上限ルール
  - パス反映失敗時の復元導線
  - パス加入状態の確認導線（`クォーツ購入画面 > 月額パス`）
  - アイテム別有効期限（30日/180日/無期限）

### Should
- 味方行動順序をルールエンジン化（行動種別 -> 編成位置）
- 引き継ぎ時の安全操作チェック（タイトル戻り推奨）
- 共鳴アビリティとサポート枠の制約チェック

### Could
- ダメージ/クリティカル式を差替可能な係数テーブル化
- ヘルプ記載URL（ガイドライン等）への遷移検証をCIに追加
- 未確定ルールを feature flag 化してA/B検証
