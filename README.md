# HBR Battle Simulator

ターンごとの行動、SP、OD、バフ・デバフ、敵状態などを記録・再計算するための非公式バトルシミュレーターです。

ブラウザ上でパーティー、敵、ステージ条件を設定し、戦闘開始後は各ターンでスキルを選んでいくだけで、行動履歴と各種リソース変化を確認できます。

## 使い方

### 1. パーティーを設定する

画面上部の `Party` で使用するスタイルを選びます。

各キャラクターには、必要に応じて以下を設定できます。

- 装備スキル
- 初期SP
- サポート枠
- 属性ブレスレット
- その他の初期状態

よく使う編成はプリセットとして保存できます。

### 2. 敵を設定する

`Enemy` で敵を選択します。

敵数、耐性、吸収、Eシールドなどを設定できます。

テンプレート敵を選んだあと、必要に応じて手動調整してください。

### 3. ステージ条件を設定する

`Stage` でステージ効果や初期ギミックを設定します。

OD/SP 系の補正、ターン開始時効果、フィールド系の初期状態など、対応済みの項目をここで指定できます。

### 4. 戦闘を開始する

設定が終わったら戦闘を開始します。

以降はターン行で前衛キャラクターのスキルを選択していきます。

スキルを選ぶと、SP、OD、状態変化、敵状態などがプレビューされます。

### 5. ターンを確定する

内容に問題がなければターンを確定します。

確定した行は履歴として残り、後から見直しや一部編集ができます。

## 操作の補足

### キャラクター詳細

キャラクターアイコンを右クリックすると、詳細ポップアップを開けます。

確認できる内容:

- 状態変化
- アビリティ
- パッシブ
- フィールド
- 威力詳細

`威力詳細` では、現在の行動に対して採用されているバフ・デバフ・相性の内訳を確認できます。

### 敵情報

敵名や敵情報エリアをクリック / 右クリックすると、敵詳細を確認できます。

確認できる内容:

- 耐性・吸収
- Eシールド
- Break / Down 状態
- 防御ダウン、脆弱、耐性ダウンなどの状態
- 手動 Break / Kill / HP Break 操作
- 召喚や敵スロット関連の操作

### ターゲット指定

単体スキルでは、敵ターゲットや味方ターゲットを指定できます。

敵が複数いる場合や、手動で Break 対象を合わせたい場合は、ターゲット指定を確認してください。

### 追撃・特殊操作

追撃、総攻撃、魔界騎兵起動、フォームチェンジなど、一部の特殊操作はターン行の操作チップやボタンとして表示されます。

表示されている操作は、その時点の状態や条件に応じて有効化されます。

### OD / 追加ターン

OD を発動すると、OD 行動として記録されます。

追加ターンが発生するスキルを使った場合は、追加ターン用の行が作成されます。

OD ゲージがマイナスになるケースも扱います。

## 保存と読み込み

現在の状態は JSON として保存できます。

- `JSON保存`: 現在のセッションを保存
- `JSON読込`: 保存済みセッションを復元
- `PNG保存`: ターン一覧を画像として保存

## ローカル起動

```bash
npm install
npm run dev
```

表示されたローカル URL をブラウザで開いてください。

## 開発用コマンド

```bash
npm test
npm run lint
npm run test:e2e
```

## ディレクトリ構成

```text
ui-next/       現行UI
src/           計算・ドメインロジック

tests/e2e/     Playwright E2E
json/          読み込みデータ
docs/          設計メモ・実装計画

scripts/       開発・分析用スクリプト
```

## 注意事項

このツールは非公式のシミュレーターです。

すべての挙動を完全再現するものではなく、一部のスキル、装備、敵行動、ステージ効果は未対応または近似実装です。

計算結果は確認・検討用として扱ってください。
