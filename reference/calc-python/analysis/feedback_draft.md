### 【確認された不具合と再現手順】

#### 1. 脆弱の計算漏れ（通常期待値テーブル ＆ 逆算用AS列）
* **仕様上の前提（正常な動作）**:
  * 脆弱の合計値（`AJ94`）の数式 `=if(AJ88>100%, ..., 0%)` が示す通り、敵に弱点（`AJ88 > 100%`）が存在する場合のみ脆弱が有効化される仕様は、ゲーム内の仕様に即した意図通りの挙動であることを確認しております。
  * また、詳細計算エリアの各ヒットごとの計算セル（`AZ266`〜`AZ506`）においては、脆弱を含むデバフ合計セルである `$BI$262`（`=sum(AJ71:AJ78)+AJ81+AJ94`）が正しく参照されているため、弱点属性時に脆弱のON/OFFを切り替えると、最終ターン合計ダメージ（`AZ33`）には正しく反映されます。

* **不具合内容（期待値テーブルおよび逆算セルでの反映漏れ）**:
  弱点属性が有効であり、かつ脆弱デバフが有効化されている状態であっても、以下の箇所で脆弱（`AJ94`）が計算から漏れてしまっています。
  1. **通常期待値テーブル（Y列・AB列、Y12〜Y570等）での漏れ**: 通常ダメージ期待値の計算式において、脆弱（`AJ94`）が掛けられていません（`AJ82`（防御デバフのみ）のみが掛けられています）。このため、弱点属性かつ脆弱ONであっても、画面左上の通常期待値（`W4`）やクリティカル期待値（`W10`）には脆弱の効果が一切乗りません。
  2. **逆算用セル（`AS262` / `AS263`）での漏れ**: 通常期待値テーブルからベース値を逆算する数式において、除算部に脆弱が含まれていません。

* **再現手順**:
  1. 敵の弱点設定を行い、雷属性耐性を弱点（例: 600%）に設定します（`AJ88 > 100%` となる状態）。
  2. 「脆弱1」のON/OFF（`AF90`）を「TRUE」にし、脆弱合計（`AJ94`）に効果量を発生させます。
  3. この状態で、画面左上の通常期待値（`W4`）および右側の最終ダメージ結果（`AZ33`）の数値を確認します。
  4. その後、脆弱（`AF90`）を「FALSE」に切り替えます。
  5. **結果**: 右側の最終ダメージ結果（`AZ33`）は脆弱のOFFに応じて減少しますが、左上の通常期待値（`W4`）は**数値が1ダメージも変化しません**。

* **該当箇所の数式例**:
  - `Y22` (期待値): `=W22*($AJ$88)*($AJ$10)*($AJ$11)*($AJ$82+100%)*($AJ$67+100%)*($AS$42+100%)`
    *(デバフ乗算部に `AJ94`（脆弱）が含まれていません)*
  - `AS262` (詳細計算逆算元): 脆弱を含まない `($AJ$82+100%)` で除算・逆算を行っています。

* **修正方針**:
  - 通常期待値テーブル（Y列/AB列）: デバフ乗算部 `($AJ$82+100%)` の箇所を、脆弱を加算した `($AJ$82+$AJ$94+100%)` に修正します。
  - 詳細計算逆算元（`AS262`/`AS263`）: 脆弱を含まない `($AJ$82+100%)` で逆算している箇所を、正しく `$BI$262` を用いた `($BI$262+100%)`（または `$AJ$82+$AJ$94+100%`）に置き換えます。

#### 2. 詳細計算エリアにおける連撃の加算バグ（BK266〜BK269）
* **不具合内容**:
  右側の詳細ダメージ計算エリア（BK266〜BK269）において、連撃効果量（`BK264` または `BK265`）が攻撃バフやフィールド、心眼と同じカッコ内で「加算」されてしまっています。
* **該当箇所の数式例** (例: `BK266`):
  `=BK262*(100%+SUM(AJ47:AJ49)+AJ52+AJ59+AJ34+AJ41+AJ45+AS32+AX49%+BK264)`
* **現在の計算構造**:
  `ダメージ = 基礎ダメージ × (1 + スキル攻撃バフ + 心眼 + フィールド + 連撃)`
* **修正方針**:
  連撃は他のスキル攻撃バフ（スキル攻撃力アップ枠等）とは独立したカテゴリであり、「乗算」の関係になります。
  そのため、攻撃バフ枠のカッコ内から `BK264`（ブレイクピアスの場合は `BK265`）を除外し、別枠で `*(100% + BK264)` を乗算するように修正してください。
  （正しい計算構造: `ダメージ = 基礎ダメージ × (1 + スキル攻撃バフ + 心眼 + フィールド) × (1 + 連撃)`）
